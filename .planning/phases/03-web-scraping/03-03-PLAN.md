  <action>
    Edit src/job_mailer/__main__.py:

    1. Add imports at the top:
       - `import csv` (stdlib)
       - `from job_mailer.scraper import scrape_company`

    2. Replace the stub `typer.echo(f"Config loaded. Input file: {input}")` line with:
       ```python
       typer.echo(f"Config loaded. Processing: {input}")
       with open(input, newline="") as fh:
           reader = csv.reader(fh)
           for row in reader:
               if not row or not row[0].strip():
                   continue
               url = row[0].strip()
               if url.lower() == "url":
                   continue  # skip header
               try:
                   record = scrape_company(url)
                   email_display = record.email_found or "no email"
                   typer.echo(f"  {record.company_name} — {email_display} — {record.status.value}")
               except Exception as exc:
                   typer.echo(f"  WARNING: failed to scrape {url}: {exc}", err=True)
       typer.echo("Done.")
       ```

    Note: The CSV expected format is a single-column file with URLs. No header is required. A header row with "url" (case-insensitive) is skipped if present. Later phases will add additional columns.
